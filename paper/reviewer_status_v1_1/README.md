## Main conclusion

The context-bottleneck experiment shows that the earlier context-mediation failure was substantially architectural: the standard router used a direct sensory shortcut, while forced context routing remained competitive and produced a specific causal route direction. However, the current geometry objective improves route order rather than context-space order, does not improve prediction, and can increase local route order while degrading global centroid alignment.
